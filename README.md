# **Проектная работа "Путешествие по России"**  
## Описание проекта:  
Адаптивный сайт для просмотра на экранах от 320 пикселей до 1280 пикселей, созданный по макету Figma о путешествиях по России.  
## Технологии:  
* Grid Layout
* Flexbox
* Макет Figma
* Медиазапросы
* Подключение шрифтов
